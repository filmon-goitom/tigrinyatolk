# TigrinyaTolk Static Website

A responsive single-page website built with Tailwind CSS.

## 🚀 How to use
1. Upload this folder to your GitHub repository.
2. Go to **Settings → Pages → Branch: main → / (root)** → Save.
3. Your site will be live at: `https://yourusername.github.io/tigrinyatolk/`

## 📁 Structure
- `index.html` → main page
- `assets/images/` → put your images here (e.g. hero.jpg, interpreting.jpg, team.jpg)
- `assets/style.css` → optional custom CSS
